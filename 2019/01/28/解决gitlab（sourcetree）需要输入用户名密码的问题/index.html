解决gitlab（sourcetree）需要输入用户名密码的问题

2019-01-28

每次pull或者push时都要输入好几次密码，真的很烦。下面的方法亲测有效，希望能帮到你！

1. 进入~用户目录（不是工程目录哦~）：cd ~

2. 建立文件： touch .git-credentials

3. 编辑文件： vi .git-credentials

4. 添加http://用户名:密码@gitlab.com

5. 执行命令：git config –global credential.helper store

6. 查看文件：more .gitconfig，可以看到如下信息，设置成功
